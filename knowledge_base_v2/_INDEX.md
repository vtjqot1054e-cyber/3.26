## 向量检索（Smart Connections）

- 已在 `vault/.obsidian/plugins/smart-connections/` 安装插件文件（`manifest.json`、`main.js`、`styles.css`）
- 已在 `vault/.obsidian/community-plugins.json` 启用 `smart-connections`
- 首次使用：打开 Obsidian 后等待索引完成，向量目录位于 `vault/.smart-env/`
- Git 同步要求：根 `.gitignore` 不可忽略 `.smart-env/`，否则两地无法复用向量索引
