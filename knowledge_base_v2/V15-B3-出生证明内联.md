# 出生证明内联是三元结构的手动版

## 内涵

V15-B2 说结论和证据之间没有桥。修复方式：**每条结论旁边挂出生证明**——用户原话 + AI判断 + 碰撞过程 + 成立条件 + 时间锚点，内联进条目里。

**核心逻辑**：不需要知识图谱，不需要向量库，就是在 refined 条目里加原始证据。补全后的条目已经隐含实体、关系、属性——这本身就是三元结构，只是手动维护。

**边界**：等条目积累到手动维护吃力时，才是三元结构/知识图谱真正需要上的时机。现在强行上是建设癖。
