# A15：底层人格模式——镜像效应

## 结论

用户的底层人格存在"镜像效应"——用别人当年让自己不平衡的方式，去对待现在的合作者。

## 镜像链条

### 第一层：金马→新媒体公司

| 你在金马的感受 | 你给B和C的感受 |
|---------------|---------------|
| 我干活，股东分钱——不平衡 | 我出钱要固定分红——他们不平衡 |
| 大股东/四股东"躺着拿钱" | B和C觉得你是"出钱想躺着拿"的人 |
| 贡献≠回报 | 他们觉得贡献≠回报 |

### 第二层：金马→你和员工

| 你对四股东的感受 | 员工对你的感受 |
|----------------|---------------|
| 没业绩但想管我 | 甩手老板但还要分钱 |
| 制度不接地气 | 框架不接地气 |
| 定任务、监控我 | 讲大道理、要我们改变思维 |

### 第三层：金马→你和张梦凡

| 你在金马的感受 | 你给张梦凡的感受 |
|---------------|-----------------|
| 我干活，股东分钱——不平衡 | 她出资源，你们赚钱——不平衡 |
| 别人用我的劳动成果 | 你们用她的平台赚钱但不分享 |

## 用户自省

> "这可能是我的底层人格就这样"
> "我可能比她更狠，因为我是草根出生可能未必有她的心胸"

## 模式识别

### 贡献-回报失衡的循环

```
第一阶段：你是"干活的"，大股东/三股东是"分钱的"——你不平衡
    ↓
第二阶段：你开单100%，四股东没业绩但想管你——你更不平衡
    ↓
第三阶段：你是"出钱的"，B和C是"干活的"——他们不平衡
    ↓
现在：你是"甩手老板"，员工是"干活的"——员工不平衡
```

### 制度-执行冲突的轮回

```
第二阶段：四股东给你定制度、定任务、监控你——你反感
    ↓
现在：你给员工定框架、讲大道理、要他们改变思维——员工反感
```

## 北京公司的区别（用户声明）

> "北京公司我会全力以赴写方案，做系统，做架构，不会甩手不管"

与新媒体公司的本质区别：
- 新媒体公司：投资人思维、要固定分红、甩手
- 北京公司：建设者思维、承担主要责任、不甩手

**但"朝令夕改"和"怕"还在——需要转化层机制来对冲。**
