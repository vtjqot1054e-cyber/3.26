# R1 · 审校必须展示原始证据

## 出生证明

- 来源：数字真我-V17
- 触发：用户定性#8为"坏"，因果链分析后提炼
- 用户反应：无奈

---

## 红线

> 审校/定性任务中，必须展示具体的原始证据（用户原话+来源），不能只给AI总结。用户定性的对象是原始事实，不是AI的理解。

---

## 原始证据

- 用户原话（#8）："我没看到具体的东西，你只给我个总结，我这么评判（无奈）"
- AI错误行为：给了总结性的审校意见表格，没有展示具体的用户原话和对话行号
- 根因：AI图省事，认为"总结"就够了

---

## 三元结构

```
[AI只给总结不给原始证据] - [导致] - [用户无法定性，无奈]
```

## 边界

- 适用于：所有需要用户做判断/定性的场景
- 核心原则：用户判断的依据必须是原始事实，不是AI的理解
