# 原料收件箱

所有原始素材统一丢这里，不用整理，格式不限。

## 支持的来源
- 钉钉转写（导出的 txt/md 文件）
- 微信聊天（复制粘贴成 txt）
- AI 对话片段
- 视频字幕（vCaptions 导出的 txt）
- 网络文章/报告
- 任何觉得有用的文字

## 钉钉会议文件格式（推荐）

如果有钉钉转写 + 钉钉摘要，请按此格式放在同一个文件里：

```
## 原始转写
（在这里粘贴钉钉的语音转写原文）

## 钉钉摘要
（在这里粘贴钉钉AI生成的摘要，可选）
```

> **重要**：AI 清洗只会读取"原始转写"部分。钉钉摘要仅保留给你人工对照参考。

## 其他来源

直接丢文本文件进来即可，不需要特殊格式。

## 文件命名（推荐加前缀）

文件名前缀决定清洗产出的 `source` 标签：

| 前缀 | 来源标签 | 举例 |
|------|---------|------|
| `钉钉-` | 钉钉转写 | `钉钉-项目周会.md` |
| `视频-` | 网络视频 | `视频-某某演讲.txt` |
| `微信-` | 微信聊天 | `微信-和刘哥聊天.txt` |
| `语音-` | 语音转写 | `语音-车上随想.md` |
| `随想-` | 随手笔记 | `随想-合作模式思考.md` |
| `文章-` | 网络文章 | `文章-行业报告.txt` |
| 无前缀 | 其他 | `whatever.txt` |

> 不加前缀也能处理，只是产出标签会显示"其他"。

## 规则
- 丢进来就行
- N8N 每 30 分钟自动扫描处理
- 处理完的原料移到 `已处理/` 目录（保留备查）
- 清洗结果出现在 `碎片暂存/`
