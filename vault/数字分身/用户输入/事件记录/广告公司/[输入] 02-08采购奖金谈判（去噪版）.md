# 02-08 采购奖金谈判（去噪版）

## 一、背景

杨红（采购）认为年终奖少了，曾锐（主管）让她直接来找老板。老板对此不满，召集三方当面对质。

---

## 二、核心矛盾

### 1）曾锐推责
- **我的批评**：你是部门负责人，权力给到了你就要解决问题。你没摆平都来找我，算什么意思？
- **曾锐回应**：采购就按照以往的额度划分。（言下之意：他不想担责）

### 2）杨红的诉求
- **现状**：按比例算是15000
- **诉求**：希望增加到20000
- **理由**：
  - 对账量增加
  - 标书商务部分做得多
  - 采买事务增加
  - 徐芳做了很多工作，需要调配空间

---

## 三、奖金机制确权

### 基数确定
| 项目 | 数值 |
|------|------|
| 采购/财务基数 | **0.5%**（按毛利计算） |
| 2024年毛利 | 28万 |
| 2024年应发 | 14000 |
| 2025年毛利 | 31万 |
| 2025年应发 | 15500 |
| 增量 | 3万 |
| 人均增长 | 3400（9人分配） |

### 最终结果
| 角色 | 金额 | 说明 |
|------|------|------|
| 杨红（采购） | **17000** | 14000基数 + 3000调配 |
| 财务（会计） | 15000 | 同比例，比去年18000少 |
| 徐芳 | 10000 | 杨红分配给她的 |

### 我的判断
- 财务工作量没增加（只是多开几张发票）
- 采购工作量确实增加（投标报名、供应商对账）
- 杨红在人均增长（3400）基础上略有上浮是合理的
- 但不能超太多，否则其他人会不满

---

## 四、杨红的综合收入结构

| 来源 | 金额 | 说明 |
|------|------|------|
| 广告公司采购奖金 | 17000 | 本次确权 |
| 会务采购基数 | 46000 | 未增长（半年脱产） |
| 会务采购增长 | 14000 | 补上去年差的4000+今年的 |
| 滴答滴分成 | 约15000 | 周末做项目 |
| **综合收入** | 约77000 | 比去年65000增长 |

### 关于会务采购
- 杨红半年时间被老板调离采购岗位
- 这半年由徐芳承担工作
- 杨红主动提出增长，被老板否决
- 老板的理由：你没做，不能要增长；徐芳做了，但那是你带的人，功劳算你的

---

## 五、老板的管理思考

### 关于量化
> "你能不能量化出来？做一本标书好多钱，准备一份资料好多钱，就像供应商一样签年度协议。明确单价、明确工作量、做了好多算好多。"

- **目的**：避免每年扯皮
- **好处**：摸鱼的人不存在了，不用担心谁想不想做
- **风险**：如果杨红量化不好，可能拿得更少

### 关于保护伞
> "现在这种方式，说个难听的话，你是在我的保护伞之下的。你量化不好你玩到亏。"

- 杨红自己也承认："一量化我就可能拿不到那么多"
- 老板的态度：所以别闹，该感谢的要感谢

---

## 六、关于会务公司的分配问题（延伸讨论）

### 增长分配不均
| 现象 | 原因 |
|------|------|
| 总量增长31000 | 毛利从28万到31万 |
| 部分人不增长甚至倒退 | 新人进来分走了增量 |
| 设计停滞 | 增长部分贡献给了薛琳 |
| 何云刚倒退 | 团团进来后，总量不变，人多了 |

### 老板的批评
> "他们回不来的本质问题是不会当领导，不知道怎么当领导。没有本着帮助别人成功的心态在做事。"

- 哪个有空安排哪个
- 哪个用得顺手喊哪个
- 想照顾的人反而没给安排活路
- 结果：发钱时才发现分配失衡

---

## 七、会议结论

### 本次确定
1. 杨红广告公司采购奖金：17000（确权）
2. 杨红会务采购：46000（不增长）
3. 补去年差额：4000
4. 徐芳：10000（杨红自己分配）
