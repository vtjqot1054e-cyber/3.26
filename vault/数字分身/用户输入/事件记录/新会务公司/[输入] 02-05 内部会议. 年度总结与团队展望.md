# 02-05 年度总结与团队展望内部会议

## 会议性质
年度总结会，每人轮流发言，老板最后做总结讲话。

---

## 第一段：牟成发言

**语义转写**：
牟成自述全年项目量少（年前一个、年尾一个），关键词是"热情"——通过主动沟通、积极反馈、行动示范来影响他人。

**画像信号**：
- 发言量少、内容空泛
- 自我定位：正能量、主动沟通

---

## 第二段：钱程（薛白）发言

**语义转写**：
钱程被问"跟同事沟通中有没有挖到宝藏"，他承认"跟同事之间确实挖到的比较少"，但跟客户和供应商有：

1. **客户端**：沟通时带一句"其他部门有没有需求"
2. **同行端**：贵州项目帮同行排日程，额外收钱
3. **供应商端**：异地看场地时收集搭建供应商信息
4. **创新案例**：帮金马邱旭项目利用AI自己搞了个"假的试片室"，赚了一万

**总结金句**：
> "首先你得清楚你能做到哪些，第二你需要哪些"

**画像信号**：
- 有商业敏感度（能发现机会）
- 但承认跟同事沟通挖宝少 → 内部协作弱
- 愿意分享方法论

## 第三段：刘睿发言

**语义转写**：
刘睿自述项目多、成就感强，印象最深的是梅奥项目——核心成员只有四五个人，把会做完了。

**亮点**：客户带着团队一个一个介绍"这个人在项目里干了什么"。

**对团队的评价**：
> "去年我们团队没有一个人是比较拖后腿的"
> "有几位成员的进步也很大"

**对公司的吐槽**：
1. **迟到问题**："大家不要太迟到，不要太拖时间，没有时间观念"
2. **饭局中途走**："跟客户吃饭的时候不要中途走"

**画像信号**：
- 以团队成果为荣
- 敢于当众提出批评（迟到、中途走）
- 对下属有保护意识（"没有拖后腿的"）

## 第四段：杨红发言

**语义转写**：
杨红坦诚全年状态：**上半年焦虑，下半年迷茫**。

**上半年**：
- 做后勤（公司大管家）
- 最焦虑的事：做标书
- 成就：两个标书没靠关系中标了

**下半年**：
- 从后勤转到广告项目组
- 感觉迷茫、失去自信、"失去了曾经的自己"
- 但从钱程那里接的口腔客户广告需求得到肯定

**自我定位**：
> "我更多的时候是在一个助力者的位置，不是一个主导者"

**画像信号**：
- 自我认知清晰（知道自己不是主导者）
- 有成长意愿（想多学）
- 但安全感不足（迷茫、失去自信）

## 第五段：张兰心发言

**原文**：
> "其实刚才你们也帮我回忆了很多，还是各种各样的。在一年当中，就是可能大家一起努力，然后攻下了一个一个难关吧，就是在攻下了之后就会觉得挺有成就感的。"

**语义转写**：
张兰心被问"喜欢什么样的工作环境"和"如何创造喜欢的工作环境"。

**关于喜欢的工作环境**：
> "我现在这个工作环境我就觉得挺喜欢的，因为大家基本上都已经是比较熟悉的，就比较熟悉大家能够做到哪种程度，然后也知道就是他的缺陷在哪儿"

**关于创造喜欢的工作环境**：
> "我觉得这个事情是相互的吧，就是你如果是坚定的要在团队里面继续待着的话，就是不止你自己要努力的学会改变这个团队，就是你要适应团队，团队也要适应你，我觉得这个是双向的"

**关于金牛座**：
> "我并不抠门，甚至守着很漏财"

**关键结尾**：
> "然后其他的问题我就不回答了哈"

**画像信号**：

| 信号 | 解读 |
|------|------|
| 喜欢"熟悉的"环境 | **安全感需求强**——熟悉=可预测=安全 |
| "知道他的缺陷在哪儿" | 她在**观察和记录**每个人的弱点 |
| "你要适应团队，团队也要适应你" | **表面双向，实际是要求团队适应她** |
| "其他问题不回答了" | **选择性回避**，控制信息输出 |

**上帝视角**：
张兰心的"喜欢熟悉环境"不只是安全感，更是**信息优势**——她熟悉每个人能做到什么程度、缺陷在哪儿，这让她在组织中有控制感。这与她"故意垄断知识"的画像一致。

## 第七段：老板讲话（核心）

### 7.1 业绩汇报

| 业务线 | 2025年表现 |
|--------|-----------|
| 会务团队 | 净利润增长40% |
| 运营前沿团队 | 由亏损转正值 |
| 广告团队 | 净利润稳中有升 |
| 视频小组 | 到了价值回归阶段 |

> "四条线没有一条是后退的，不是运气，是在座各位的努力"

### 7.2 老板自我介绍（首次正式）

**真名**：何大为（"大有作为"）
- 平时用的签字名和外面名字不一样
- 入行19年，从"小鲜肉熬成老腊肉"

**早期经历**：
- 搬过设备、搭过屏幕、调过投影
- 跟供应商一起弄背景板
- 现场美工问题自己能做
- 写过宣传稿、领导发言稿
- 七八米高架子上修设备

> "当时没想过有一天能够站在这个地方跟大家分享"

### 7.3 老板的自我认知

**不擅长的**：
1. **KPI管理**："办公室里出来的KPI，我是底层代码反感，完全不能接受"
2. **应酬饭局**："酒量不好，一喝就醉，一醉就失态，一失态就断片"
3. **打关系**："不是那两个领域里面的料"

**擅长的**：
1. **透过现象看本质**："异常的"
2. **梳理复杂项目和人际关系**：细节把握好
3. **把钱花在该花的地方**："没有办公室，但业绩不靠办公室"
4. **培养人**："从来没有放弃过认真做事的每一个人"

> "只要你还在认真做事，只要你还在正路上，无外乎就是陪着你耗"

### 7.4 关于竞争对手

> "这些徒弟当中有一些已经成为了竞争对手，但我不怕教你"
> "能带出厉害的人，这正是师傅的本事"
> "我相信他们对我的风评比较好"

### 7.5 创业初心

> "五年前决定从金马重新再开"
> "原因非常简单：我就想让跟着我做事的人能够挣钱，挣更多的钱"

**投资回报数据**：
- 合伙人两年前投入60万
- 两年最低收益146万（不含送的14万）
- 含送的：160万

### 7.6 各岗位总结（老板原话）

| 岗位 | 要求 |
|------|------|
| 市场扩展 | 货能出去，赔得下来 |
| 客户经理 | 特别要细节，成为去仓库 |
| 项目经理 | 扛得住压力，顶得上现场 |
| 设计排版 | 接得住想法 |
| 运营策划 | 出得了成果，经得起推敲 |
| 采购 | 守得住底线，压得下成本 |

### 7.7 对未来的判断

**会变的**：政策、市场、客户需求

**不变的**：
1. 认真做事的人永远被需要
2. 老板对质量的要求不会变
3. 对团队的承诺不会变
4. 仰望星空的态度不会变

> "外面的世界是不确定的，但站在这个地方的人是确定的"
