# 同步方案

> 在公司和家里保持数字分身同步

---

## 推荐方案

### 方案1：OneDrive / 坚果云（推荐）

**步骤：**
1. 把 `E:/数字分身/` 文件夹移动到 OneDrive（或坚果云）同步目录
2. 在两台电脑上都安装同步客户端
3. Obsidian 打开同步目录下的库

**优点：**
- 免费
- 自动同步
- 版本历史

**注意：**
- 避免同时在两台电脑编辑同一文件

---

### 方案2：Git 同步

**步骤：**
1. 在 `E:/数字分身/` 初始化 Git 仓库
2. 推送到 GitHub（私有仓库）
3. 在另一台电脑 clone

**命令：**
```bash
cd E:/数字分身
git init
git add .
git commit -m "初始化数字分身系统"
git remote add origin [你的仓库地址]
git push -u origin main
```

**优点：**
- 完整版本控制
- 可以看到每次变更

**缺点：**
- 需要手动 push/pull

---

### 方案3：Obsidian Sync（付费）

**步骤：**
1. 订阅 Obsidian Sync（$8/月）
2. 在设置中开启同步

**优点：**
- 无缝集成
- 端到端加密
- 实时同步

## 冲突处理

如果同步出现冲突：
1. Obsidian 会保留两个版本
2. 手动比对，保留正确的
3. 删除冲突文件
