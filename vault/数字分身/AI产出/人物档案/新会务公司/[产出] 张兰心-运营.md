## 深层分析（2026-01-24对话验证）

### 核心：没有安全感

### 和刘的关系
- 好朋友
- 刘生病主动买药
- 安排出去玩
- 说明：张不是对所有人都打压，是选择性的

### 为什么讨厌薛（用户分析）
1. 工作没做好看不起
2. 薛搞定了客户，张没有 → 嫉妒+威胁
3. 老板喜欢这样的人 → 作为领导会打压

### 个人情况
- 五官好，但胖和矮
- 没看到有人追求
- 家庭条件还可以
- 加班熬夜没问题
- 不像老板那么在意钱

### 她在意什么
- 被需要、被认可、归属感
- 不可替代 = 安全感

### 管理原则
| 原则 | 做法 |
|------|------|
| 给归属感和认可 | 老板多认可她 |
| 不让她直接面对薛 | 薛的报账通过老板协调 |
| 不管"打压" | 她对薛的态度不介入 |
| 让她和刘绑定 | 强化安全区 |
| 两个大项目 | 暂时不逼她丢出来 |

### 职能设计
- 运营 + 策划（老板给框架，她填充）+ 审计
- 可行，但需要老板支撑

### 职责边界（2026-01-26确认）

| 负责 | 不负责 |
|------|--------|
| "怎么投"（具体应答、围标策略） | "该不该投"（商业判断、报价） |
| 用逻辑判断用哪些公司、为什么 | 市场感觉、价格判断 |
| 输出执行指令单 | 决定投不投 |

**与成的配合**：成做商业判断（投不投、报价、围几家），张用逻辑兜底（哪些公司、怎么应答）
