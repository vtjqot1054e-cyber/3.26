## 一、财务人员奖金分层逻辑

### 结论

财务人员分两类，奖金逻辑不同：

| 类型 | 特征 | 奖金逻辑 |
|------|------|----------|
| **会计出纳** | 拿底薪+双薪，业绩增长跟他们关系不大 | 固定奖金 |
| **财务经理/总监** | 深入参与业务，帮公司筹划节税 | 按节约额度提成（约10-11%） |

### 因果链

- **现象**：公司业绩增长，但普通财务人员工作量没变
- **根因**：不同层级的财务人员价值贡献不同
- **改变路径**：区分定位，匹配奖金逻辑

### 刘琦案例

| 年份 | 解决额度 | 节约税收 | 奖金 | 比例 |
|------|----------|----------|------|------|
| 2024 | 310万 | 约31万 | 3.5万 | 约11% |
| 2025 | 410万 | 约32万 | 3.5万 | 约11% |

> **注意**：刘琦能节税的关键是**她家里开厂，能提供成本票**，这是资源优势，不是单纯的专业能力。

## 二、四流合一与项目核算

### 结论

公司必须做到四流合一，每个项目单独核算。审计不看总账，看每一笔项目线索。

### 四流合一

| 流 | 说明 |
|-----|------|
| 发票流 | 发票的开具和接收 |
| 资金流 | 银行转账记录 |
| 合同流 | 合同签订和履行 |
| 货物/服务流 | 实际交付的货物或服务 |

四流必须匹配一致，不能"票是A的，钱给B，合同签C"。

### 当前问题 vs 未来要求

| 当前 | 未来 |
|------|------|
| 平时不管细节，年底算总账 | 每个项目单独核算 |
| 审计来了拿不出细账 | 能解释每一笔交易流水 |
| 费用乱挂账（如买酒） | 转化到项目合理开支 |

### 操作方法

| 问题 | 解决方案 |
|------|----------|
| 供应商月结导致难以匹配项目 | 让供应商提前开票，账先结，付款后置 |
| 费用不合规（如买酒） | 转化到项目的合理开支里，不能单独挂账 |
| 年底突击开票 | 高风险，必须改为项目实时结账 |

### 外部压力

> **学会反腐又来了**——趁现在没出事，把账理顺。

## 三、员工奖金个税筹划方案

### 结论

奖金超3.6万的员工，可通过名下个体户开票避税（10%→1-3%）。

### 方案逻辑

| 步骤 | 说明 |
|------|------|
| 1 | 员工用名下个体户开票给公司 |
| 2 | 公司转账到个体户（有票入账） |
| 3 | 员工实际税负从10%降到1-3% |

### 操作原则

| 原则 | 说明 |
|------|------|
| **公司不倒贴** | 3%的税点从员工奖金里扣，不是公司额外支出 |
| **有渠道的自己走** | 有个体户的员工自己开票 |
| **没渠道的按规矩** | 找不到票就交税，公司不承担这个责任 |
| **互惠互利** | 帮员工省税，员工也要帮公司解决票据问题 |

### 谁有个体户

| 人员 | 渠道 |
|------|------|
| 杨（杨红？） | 有公司账户 |
| 周学林 | 有个体户 |
| 狄大帝 | 有个体户 |
| 其他人 | 自己想办法找票，找不到就交税 |

### ⚠️ 风险提示（2026-04-03 用户补充）

> **今年国家会加大对个体户开票问题的稽查！**

| 风险 | 说明 |
|------|------|
| 虚开发票 | 个体户开票必须有真实业务，否则涉嫌虚开 |
| 开对公户难 | 需要实际经营地址，银行上门拍照核实 |
| 私户走账 | 资金流向私人账户，合规性有瑕疵 |
| 稽查收紧 | 2026年国家加大对个体户开票的稽查力度 |

### 结论

这个方案可以用，但必须谨慎：
1. 必须是真实业务
2. 优先用有对公户的个体户
3. 关注政策动向，随时调整
