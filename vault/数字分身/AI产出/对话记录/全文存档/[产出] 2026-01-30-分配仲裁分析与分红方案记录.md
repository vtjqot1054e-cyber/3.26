## 分配仲裁深度侧写

### 场景
老板主持，薛和刘（大二）关于梅奥项目分成的仲裁。核心争议：64:36的分成比例。

### 薛的表现

| 维度 | 发现 |
|------|------|
| **议价能力** | 极弱——开口8个点→5→3→"那就那么吧" |
| **自我设限** | "我要多了他也不可能答应"——还没谈就先投降 |
| **细节掌控** | 弱——自己做的事说不清楚，反复"我不晓得" |
| **交付质量** | 照片缺20-30个，劳务信息不全 |
| **归因模式** | 外归因——"他没有整理呀"、"他并没有用" |
| **理论匹配** | **习得性无助**——长期"做不好→被批评→放弃争取"的循环 |

### 刘的表现（通过老板转述）

| 维度 | 发现 |
|------|------|
| **沉默策略** | 不直接发言，让老板代为转述 |
| **边界刚性** | 同意加1个点，但扣2个点（没来上日程） |
| **惩罚导向** | 用扣分代替沟通解决问题 |
| **理论匹配** | **消极攻击**——不直接表达不满，通过间接方式表达 |

### 老板的仲裁风格

| 维度 | 表现 |
|------|------|
| **引导式追问** | "你到底做了啥子"、"具体是什么" |
| **框架先行** | "我们先按对半基数来，看多做了什么" |
| **数据思维** | 35000报价、64:36分成、48+30+22构成 |
| **教练角色** | 教薛怎么谈判 |
| **用制度解决人的问题** | "分家"、"双30%"来破局 |
