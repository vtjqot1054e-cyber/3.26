# 首届国际运动医学大会

## 战略定位

### 表面定位

| 维度 | 内容 |
|------|------|
| 我们的角色 | 学术服务商（演戏） |
| 核心目标 | 长期吃19万/年，把戏演好 |
| 本质 | 为套补贴项目提供学术外壳 |

### 真实定位：假戏真做

> **三亚项目不只是演戏拿19万，而是北京公司产品的验证场**

| 表面 | 实质 |
|------|------|
| 给李国平套补贴 | 验证"帮专家建学术品牌"模式 |
| 19万服务费 | 积累300+专家关系网 |
| 演好戏 | 打磨内容策划+宣传能力 |
| 运医前沿公众号 | 验证"帮专家发声→建立依赖"的闭环 |

### 与北京公司产品的关系

北京公司产品核心："帮专家建学术品牌"

| 北京公司产品五大能力 | 三亚项目如何验证 |
|----------------------|------------------|
| 懂上层关系网 | 300+专家数据库已建立 |
| 懂各人擅长领域 | 专家档案库含学术标签 |
| 能揣摩需求 | 陈世益7年深度绑定经验 |
| 能策划产品 | 三亚项目就是策划实践 |
| 能帮他们搞钱 | 待验证 |

### 关联公司架构

| 公司 | 法人 | 与三亚项目关系 |
|------|------|---------------|
| **维顺泰** | 待定 | 三亚项目装入维顺泰，学术端载体 |
| **益方** | 钱程 | 医院端执行载体 |

> **运动医学是"吃饭的活"，老于/口腔是"建护城河的战场"。**
> 
> 但三亚项目的真正价值是**产品验证**，不只是吃饭。

这个项目的价值（重新排序）：
1. **北京公司产品验证场**——验证"帮专家建学术品牌"模式
2. **积累运动医学专家关系网**——300+专家数据库
3. **打磨运医前沿公众号**——验证帮专家发声的闭环
4. 19万稳定收入
5. 让团队学会"演好戏"

## 项目真相

### 甲方的真实动机

> **李和苗想拿补贴，想以展览为主，连续办几年**（用户原话）

| 维度 | 事实 |
|------|------|
| 主办方 | 李国平（国平运动医学研究院）+ 苗（落地运营） |
| 核心目的 | 套取海南自贸港会展补贴 |
| 商业模式 | 会议+展览叠加申请，连续办数年 |
| 第一届定位 | **不想挣钱，摸清套路** |
| 利益分配 | 分会场坛主分20% |

### 海南补贴政策（2025-2026）

| 级别 | 项目类型 | 最高补贴 |
|------|----------|----------|
| 省级 | 重点引进展览（可持续5届） | 1000万/届 |
| 省级 | 重点引进会议（可持续3届） | 300万/届 |
| 三亚市 | 国际会议（2天+） | 150万 |
| 三亚市 | 行业学术会议 | 120万 |
| 三亚市 | 高端展览（4天+） | 500万 |
| 三亚市 | 会议+展览叠加 | **最高800万** |
| 淡季 | 5-8月办会 | 额外100万 |

**我们的实际角色**：
- 不是"专家资源方"
- 是陈世益项目的"带宣传的执行团队"
- 现场讲师调整的负责人（不是大级别专家，说了来不来就换人）
- 跟陈世益的学术组织者关系好

**请专家的实际流程**：
1. 分论坛主席确定名单
2. 李国平出面请
3. 我们负责确认、调整、执行

**三亚的误解**：以为我们对运动医学资源很多、扎根很深，实际并非如此。

## 日程分工（兰心 vs 三亚）

**兰心负责的**：
- 临床技术分会场01-05（膝、肩、髋、足踝、军事医学）—— 真学术内容
- 慢病、营养、心理等"软"内容

**三亚负责的**：
- ACSM认证分会场（EPP/CEP/EP/CPT）—— 商业变现
- 运动康复分会场 —— 跟ACSM认证配套
- 老年医学分会 —— 张存泰主委刚上任，三亚在抢资源
- 培训课程 —— 收费项目
