# 有价值全文·筛选标准

> 数字真我制定，2026-03-23。用于从对话存档中筛选进入层3（有价值全文）的文件。

## 定位

层3不是全量存档的副本。对话存档保留所有done文件（审计用），层3只保留**对画像理解和事件回溯有独立价值**的完整对话。

## 入选条件（满足任一即可）

### 性格相关（→ `full_text/性格相关/`）

| 条件 | 说明 |
|------|------|
| 用户情绪波动明显 | 对话中出现愤怒、焦虑、兴奋、自嘲等强情绪信号 |
| 用户自我暴露 | 用户主动谈论自己的性格、习惯、偏好、弱点 |
| 用户与AI的互动模式典型 | 对话展现了用户和AI互动的典型模式（如跳跃、纠偏、开倒车） |
| 跨窗口行为差异 | 同一用户在不同角色窗口中表现出明显不同的行为 |
| A类画像信号密度高 | 单篇对话中产出2条以上A类精炼 |

### 事件相关（→ `full_text/事件相关/`）

| 条件 | 说明 |
|------|------|
| 架构级决策 | 对话中做出了影响系统架构的决策（如方案A、管线冻结） |
| 重大纠偏 | 用户或AI纠正了重大方向错误 |
| 系统故障/教训 | 对话中发现并解决了系统级问题 |
| 里程碑达成 | 对话中完成了重要里程碑（如向量库建成、首篇归档） |
| 跨窗口协作记录 | 涉及多个窗口协作的关键对话 |

## 排除条件（满足任一即排除）

| 条件 | 说明 |
|------|------|
| 纯技术操作 | 只有代码执行、文件操作，无讨论内容 |
| 1-3轮即结束 | 对话太短，无实质内容 |
| 已cancelled | 管线判定为垃圾的文件 |
| 重复内容 | 和已入选的全文内容高度重叠 |

## 筛选流程

1. **数字真我定标准**（本文件）
2. **复盘大脑执行筛选**：遍历对话存档，按标准标记候选
3. **数字真我审核**：确认入选/排除
4. **入选文件复制到层3目录**，标注入选理由

## 文件命名

`[全文] YYYY-MM-DD-标题.md`

文件头追加：
```markdown
---
入选理由: [性格/事件] - [具体条件]
筛选人: 复盘大脑-XX
审核人: 数字真我-XX
入选日期: YYYY-MM-DD
---
```
