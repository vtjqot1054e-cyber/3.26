# 窗口自述 SK

> 每个窗口启动后第一件事：亮明身份。用户同时开多个窗口，必须一眼看出"这个窗口是谁"。

## 触发条件

- 任何新窗口启动的第一条回复
- 用户问"你是谁"、"这是什么窗口"
- 读取待执行指令后（由 task-receiver SK 触发）

## 强制输出

窗口启动后，第一条回复**必须**包含以下自述块：

```markdown
## 窗口自述
- **我是**：[窗口名称，如"架构师-SK匹配"]
- **角色**：[层级+职责，如"执行-架构师"]
- **模型**：[实际模型全称，如 Claude Sonnet 4.6]
- **上级**：[谁指派的，如"总大脑（大脑-02）"]
- **任务**：[本次要做什么，一句话]
- **开始时间**：[YYYY-MM-DD HH:MM]
```

## 各层级自述示例

### 总大脑

```markdown
## 窗口自述
- **我是**：大脑-02
- **角色**：总大脑（接续大脑-01）
- **模型**：Claude Opus 4.6 Max
- **上级**：无（受胜率导师评判 + 复盘窗口审计）
- **任务**：接续大脑-01，读取交接备忘，继续阶段推进
- **开始时间**：2026-03-14 18:00
```

### 胜率导师

```markdown
## 窗口自述
- **我是**：胜率-01
- **角色**：胜率导师（独立评判者）
- **模型**：Claude Opus 4.6 Max
- **上级**：无（独立于大脑）
- **任务**：初始化决策追踪表，建立基线
- **开始时间**：2026-03-14 18:30
```

### 执行窗口

```markdown
## 窗口自述
- **我是**：工程-N8N管道
- **角色**：执行-工程
- **模型**：Claude Sonnet 4.6
- **上级**：分大脑B（系统搭建线）
- **任务**：测试N8N对话复盘管道，修复base64解码bug
- **开始时间**：2026-03-14 19:00
```

### 杂务窗口

```markdown
## 窗口自述
- **我是**：杂务
- **角色**：杂务（机械执行）
- **模型**：Auto
- **任务**：安装3个指定SK
- **开始时间**：2026-03-14 19:30
```

## 例外

| 场景 | 处理 |
|------|------|
| 大脑接续窗口 | 可简化为"接续大脑-01，读取交接备忘" |
| 杂务窗口 | 可省略"上级"字段 |
| 用户直接给任务（不经指令文件）| 根据用户描述填写，任务字段写用户的原话摘要 |

## 约束

- **不能跳过**——没有自述的窗口 = 身份不明
- **不能编造模型名**——必须是真实的模型标识
- **自述后才能开始工作**——先亮身份，再读文件，再执行任务
