# 数字分身系统 - 使用指南

> 用自然语言和AI对话，逐步养成懂你的数字分身

---

## 快速开始

### 自然语言指令

| 你说 | AI会做 |
|------|--------|
| "帮我分析一个事件" | 运行人格侧写，分析你描述的事件 |
| "帮我整理一下" | 运行碎片整合，把你的描述结构化 |
| "找一些类似的案例" | 运行知识匹配，找相关参考 |
| "检查一下系统状态" | 运行系统监控，告诉你需要更新什么 |
| "读取我的画像" | 读取核心画像，了解当前的数字分身 |
| "更新我的状态" | 更新动态状态 |

### 日常使用流程

```
1. 开始新对话时，让AI先读取你的核心画像和动态状态
2. 描述你要处理的事（可以随便说）
3. AI会根据数字分身理解你的意图
4. 对话结束时，运行系统监控，看看需要更新什么
5. 确认后，AI帮你更新相关文件
```

## 养成路径

## 注意事项

1. **确认很重要**：AI的分析需要你确认，确认后才会更新到画像
2. **碎片化OK**：你可以随便说，AI会帮你整理
3. **慢慢来**：这是养成过程，不需要一次到位
4. **物理反馈**：记得把现实世界的结果告诉AI，修正数字分身
