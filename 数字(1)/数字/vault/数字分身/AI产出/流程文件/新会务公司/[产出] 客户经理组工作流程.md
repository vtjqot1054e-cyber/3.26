## 日程管理分工

| 阶段 | 主导 | 配合 |

## 按项目阶段流程

### 1. 立项阶段

| 动作 | 产出 |
|------|------|
| 客户需求初步了解 | 需求摘要 |
| 重点分析 | 客户重点清单 |
| 内部立项沟通 | 立项确认 |

### 2. 方案阶段

| 动作 | 产出 | 协作 |
|------|------|------|
| 整合各模块方案 | 完整方案 | 运营策划出框架、市场销售出价格、项目经理出搭建 |
| 客户沟通 | 方案确认 | - |
| 方案修改 | 定稿方案 | - |

### 3. 执行准备

| 动作 | 产出 |
|------|------|
| 内部资源协调 | 资源确认表 |
| 外部供应商协调 | 供应商确认 |
| 日程敲定 | 总日程表 |
| 接待方案 | 接待分工表 |

### 4. 现场执行

| 动作 | 说明 |
|------|------|
| 接待统筹 | 现场接待总控 |
| 接收客户变化 | 第一时间响应 |
| 分发给项目经理 | 执行侧落地 |
| 突发问题处理 | 现场应变 |

### 5. 收尾阶段

| 动作 | 产出 |
|------|------|
| 项目文档归档 | 完整项目档案 |
| 项目复盘记录 | 复盘报告 |
| 客户反馈收集 | 反馈记录 |

## 对内对外工作流

```
【对外】
客户高层 ←→ 市场销售（成）：关系、情绪
客户对接人 ←→ 客户经理（刘组）：需求、进度、商务
客户执行侧 ←→ 项目经理（薛组）：执行细节

【对内】
客户经理组 ←→ 运营策划（张）：策划框架
客户经理组 ←→ 市场销售（成）：价格、关系
客户经理组 ←→ 项目经理组（薛）：执行、搭建
```
