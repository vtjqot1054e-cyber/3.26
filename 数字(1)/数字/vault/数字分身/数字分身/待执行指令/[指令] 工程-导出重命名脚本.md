# 指令：对话导出文件自动重命名脚本

> 发出方：大脑-04 | 接收方：工程窗口

## 背景

用户在 Cursor 中通过 Export Transcript 手动导出对话为 MD 文件。Cursor 默认文件名是 chat title 的 slug 形式，例如：`cursor_04_opus_4_6_max.md`。

用户需要把这些文件重命名为统一格式：`[导出] YYYY-MM-DD-HHMM-原始文件名.md`

## 需求

写一个 Python 脚本：`数字分身/brain/rename_exports.py`

### 功能

1. 扫描 `数字分身/对话导出/` 目录下的所有 `.md` 文件
2. 跳过已有 `[导出]` 前缀的文件
3. 从文件第2行提取导出日期（Cursor Export Transcript 固定格式）
4. 重命名为 `[导出] YYYY-MM-DD-HHMM-原始文件名.md`

### 日期提取规则

Export Transcript 的第2行固定格式：
```
_Exported on 2026/3/14 at GMT+8 17:36:18 from Cursor (2.6.19)_
```

提取 → `2026-03-14-1736`

### 约束

- **脚本注释和输出必须用中文**（用户需要看懂）
- Windows 环境，注意控制台编码（GBK），print 输出不要用 emoji
- 路径用相对路径，脚本位于 `数字分身/brain/`，目标目录是 `数字分身/对话导出/`
- Python 3.7+，不依赖第三方包
- 无法提取日期的文件跳过并打印警告
- 目标文件名已存在时跳过并打印提示

### 测试方法

1. 在 `对话导出/` 目录创建一个测试文件（不带 `[导出]` 前缀），内容模拟 Export Transcript 格式
2. 跑脚本，确认重命名成功
3. 再跑一次，确认已重命名的文件被跳过
4. 测试完删除测试文件
