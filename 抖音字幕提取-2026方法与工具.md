## 一、技术方案分类

### 方案1：视频OCR识别（直接识别画面字幕）

**适用**：抖音视频有硬字幕（烧录在画面上）

**原理**：
```
下载视频 → 提取关键帧 → OCR识别 → 合并文本 → 去重/时间轴对齐
```

**优点**：
- ✅ 不依赖抖音API
- ✅ 可处理任意有硬字幕的视频
- ✅ 完全本地化

**缺点**：
- ❌ 识别准确率受画面质量影响
- ❌ 需要处理字幕遮挡、特效等
- ❌ 无法获取软字幕（.srt文件）

---

### 方案2：语音转文字（ASR - 自动语音识别）

**适用**：视频无字幕，或字幕不准确

**原理**：
```
下载视频 → 提取音频 → ASR识别 → 生成字幕文件
```

**优点**：
- ✅ 不依赖原视频是否有字幕
- ✅ 可生成标准.srt格式
- ✅ 可选择多种ASR模型

**缺点**：
- ❌ 识别准确率受口音、噪音影响
- ❌ 需要GPU加速（本地部署时）
- ❌ 可能需要后期校对

---

### 方案3：抖音API/爬虫（如果有官方字幕）

**适用**：部分抖音视频有官方软字幕

**原理**：
```
获取视频链接 → 解析API → 提取字幕接口 → 下载.srt
```

**优点**：
- ✅ 最准确（官方字幕）
- ✅ 速度快

**缺点**：
- ❌ 依赖抖音API（可能需要逆向）
- ❌ 受限于反爬虫机制
- ❌ 法律合规风险

### 工作流2：OCR方案（备选）

```python
# subtitle_extractor.py
import cv2
from paddleocr import PaddleOCR
from collections import Counter

def extract_subtitle_from_video(video_path, output_path):
    ocr = PaddleOCR(use_angle_cls=True, lang='ch')
    cap = cv2.VideoCapture(video_path)
    
    fps = cap.get(cv2.CAP_PROP_FPS)
    frame_interval = int(fps)  # 每秒1帧
    
    subtitles = []
    frame_count = 0
    
    while True:
        ret, frame = cap.read()
        if not ret:
            break
        
        if frame_count % frame_interval == 0:
            # 只识别底部字幕区域
            h, w = frame.shape[:2]
            subtitle_region = frame[int(h*0.8):h, :]
            
            result = ocr.ocr(subtitle_region, cls=True)
            
            if result and result[0]:
                texts = [line[1][0] for line in result[0]]
                subtitle_text = ' '.join(texts)
                
                # 去重（相邻帧可能识别到相同字幕）
                if not subtitles or subtitles[-1] != subtitle_text:
                    timestamp = frame_count / fps
                    subtitles.append((timestamp, subtitle_text))
        
        frame_count += 1
    
    cap.release()
    
    # 写入SRT文件
    with open(output_path, 'w', encoding='utf-8') as f:
        for i, (timestamp, text) in enumerate(subtitles, 1):
            start = format_timestamp(timestamp)
            end = format_timestamp(timestamp + 1)
            f.write(f"{i}\n{start} --> {end}\n{text}\n\n")

def format_timestamp(seconds):
    h = int(seconds // 3600)
    m = int((seconds % 3600) // 60)
    s = int(seconds % 60)
    ms = int((seconds - int(seconds)) * 1000)
    return f"{h:02d}:{m:02d}:{s:02d},{ms:03d}"

# 使用
extract_subtitle_from_video('douyin_video.mp4', 'subtitle.srt')
```
